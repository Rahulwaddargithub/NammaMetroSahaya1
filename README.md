# 🚇 NammaMetroSahaya

NammaMetroSahaya is an Android application designed to assist metro passengers with easy navigation, route guidance, fare estimation, station exit information, and AI-based assistance for a better metro travel experience.

## 📱 Features

- 🚉 **Route Finder** – Find metro routes between stations
- 💰 **Fare Calculator** – Estimate metro ticket fare
- 🚪 **Exit Finder** – Get station exit/gate directions
- 🗺️ **Visual Guide** – Step-by-step metro navigation assistance
- 🤖 **AI Assistant** – Smart assistance for metro-related queries
- 👤 **User Authentication** – Login and Registration system
- ⚙️ **Settings & Profile** – User profile and app customization
- ℹ️ **About Section** – Information about the application

## 🛠️ Tech Stack

- **Language:** Kotlin  
- **IDE:** Android Studio  
- **UI Design:** XML Layouts  
- **Framework:** Android SDK  
- **Architecture:** Activity-Based Android Application

## 🚀 Installation & Setup

### Prerequisites
Make sure you have:

- Android Studio installed
- Android SDK configured
- Git installed

### Steps to Run

1. Clone the repository:

```bash
git clone https://github.com/Rahulwaddargithub/NammaMetroSahaya.git
```

2. Open the project in **Android Studio**

3. Sync Gradle files

4. Run the application using:
   - Android Emulator, or
   - Physical Android Device

## 🎯 Purpose of the Project

The main goal of **NammaMetroSahaya** is to make metro travel easier for users by providing quick access to metro routes, fare details, exit guidance, and smart travel assistance in a user-friendly interface.
